# ⚽ Lionel Messi Career Analytics Dashboard

## 📌 Project Overview

This Power BI dashboard provides an in-depth analysis of Lionel Messi's football career using match-level data. The project highlights performance trends, advanced football metrics, and competition-wise insights through interactive visualizations.

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Data Visualization
- Sports Analytics

---

## 📊 Dashboard Pages

### Page 1: Career Overview

- Total Matches
- Total Goals
- Total Assists
- Goal Contributions
- Average Rating
- Goals Trend Across Seasons
- Assists Trend Across Seasons
- Season-wise Performance Analysis

### Page 2: Advanced Analytics

- Goals vs Expected Goals (xG)
- Assists vs Expected Assists (xA)
- Key Passes Analysis
- Successful Dribbles Analysis
- Average Match Rating by Competition

---

## 📈 Key Insights

- Messi recorded 826 goals and 363 assists across 1052 matches.
- Goal contributions exceeded 1180 during his professional career.
- Consistently outperformed expected metrics (xG and xA).
- Maintained elite performance across multiple competitions and seasons.
